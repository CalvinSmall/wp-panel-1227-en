# 更新说明

## v1.2.8

### 新功能

- **防火墙 IP 搜索与分页**：安全防御页面的「当前封禁列表」新增 IP 地址搜索框，支持模糊匹配；同时加入分页功能（每页 30 条），大量封禁记录时浏览更流畅。
- **操作日志分页**：面板设置页面的「最近操作日志」新增分页功能（每页 30 条），并显示总条数统计。
- **面板数据库备份**：新增面板 SQLite 数据库备份功能，每天凌晨 2:30 自动备份，保留最近 7 份。支持手动备份、恢复和删除，恢复前自动创建安全备份。使用 `VACUUM INTO` 实现在线热备，无需停服。

### 安全与可靠性

- **数据库备份命令加固**：MariaDB 备份改为参数化执行 `mariadb-dump` 与 `gzip`，并校验数据库标识符、拒绝覆盖既有备份文件，降低命令注入和误覆盖风险。
- **面板手动更新加固**：面板更新下载增加 SHA256 与 Ed25519 签名校验、版本化备份、二进制预检和更新互斥锁，失败时尽量回滚到原二进制。
- **远程备份参数校验**：保存和测试远程备份配置时校验主机、端口、用户名、认证方式、本地备份目录和远端路径，避免非法参数进入 `rsync` 调用。
- **文件管理边界加固**：删除、复制、移动和压缩文件时补充路径边界检查，阻止删除根目录、目录复制到自身、ZIP 路径穿越和异常文件条目。
- **启动依赖行为调整**：面板启动时不再自动执行系统包安装；新装脚本已补齐 `sshpass` 安装，避免运行期隐式修改服务器环境。
- **边界测试补充**：新增数据库备份、更新辅助函数、远程备份参数校验、文件操作边界和插件接口缓存相关测试，覆盖关键安全路径。

### 开发体验

- **本地开发版本标识**：未通过发布构建注入版本号时，面板默认显示 `dev`，避免本地开发包误显示为旧正式版本。
- **Windows 验证入口说明**：补充 `scripts\verify.cmd` 说明，Windows 本地遇到 PowerShell 执行策略拦截时可直接运行该入口完成同一套验证。
